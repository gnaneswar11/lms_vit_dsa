
# C++ Library Management System

## Features
- Linked List for book storage
- Binary Search Tree (BST) for fast searching
- Mock SQL Runtime Database using HashMap
- Supports 10,000 books
- Unit Test Cases included

## Files
- library.cpp -> Main project
- test_library.cpp -> Unit tests

## Compile

### Main Program
g++ library.cpp -o library

### Run
./library

### Unit Tests
g++ test_library.cpp -o tests
./tests

## Concepts Used
- OOP in C++
- Linked Lists
- Trees (BST)
- Runtime Database Simulation
- Unit Testing
